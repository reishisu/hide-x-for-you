# HideXForYou

X (x.com) のホーム画面上部にある「おすすめ / For you」タブを非表示にし、必要に応じて「フォロー中 / Following」へ切り替えやすくする Chrome 拡張です。

> 注意: 本拡張は X Corp.（旧 Twitter）とは無関係の非公式ツールです。

---

## できること

- X のホーム上部タブから「おすすめ / For you」を見えないようにします
- 「おすすめ」が選択中の場合、可能なら自動で「フォロー中 / Following」に切り替えます
- X は SPA のため画面遷移や「戻る」で DOM が作り直されますが、再描画後も適用されるよう追従します

---

## インストール

### Chrome Web Store（公開後）
- Chrome Web Store: https://x.com/re_2521

### ローカル（開発・動作確認）
1. このリポジトリをクローン or ZIP ダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」→ このプロジェクトのフォルダを選択
5. `https://x.com/home` を開いて動作確認

---

## 使い方

1. `https://x.com/home` を開く
2. 上部タブに「おすすめ / For you」が表示されている場合、非表示になります
3. 「おすすめ」が選択中なら「フォロー中 / Following」へ切り替わる場合があります（UI 状況により変動）

---

## 権限（Permissions）

本拡張は、**X のページ上で DOM を読み取り/変更する**ために、次のホスト権限を使用します。

- `https://x.com/*`

この権限は、該当 URL パターンに一致するページで拡張が動作することを許可するものです。  
（権限は必要最小限に抑える方針です。）

## 既知の制約

- X 側の UI 変更により、動作しなくなる可能性があります
- 表示言語や A/B テスト等により、タブ検出が失敗する場合があります
- 拡張同士の競合（広告ブロッカー等）で挙動が変わることがあります

---

## 開発（Development）

- `content.js` が X の DOM を監視し、タブの再描画に追従して適用します
- 変更したら `chrome://extensions/` で拡張を「更新」し、X を再読み込みしてください
